What it is
==========
Parser for Parser for Event Information Tables (https://de.wikipedia.org/wiki/Event_Information_Table) 

Installation
============
```bash
pip install eitparser
```

Get Sources
===========
```bash
git clone https://github.com/WolfgangFahl/eitparser
cd eitparser
scripts/install
```
